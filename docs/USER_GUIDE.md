# 📖 Panduan Pengguna Umum (User Guide)

Selamat datang di Sistem Informasi Manajemen Sekolah (Sekolahku). Panduan ini memberikan gambaran umum tentang cara mengakses dan menavigasi sistem.

## 1. Akses Sistem (Login)

### Alamat Website
Buka browser (Google Chrome, Firefox, Safari) dan kunjungi alamat website sekolah Anda (Contoh: `https://sekolahku.id`).

### Halaman Login
1. Klik tombol **Masuk** di pojok kanan atas.
2. Masukkan **Email** atau **Username** Anda.
3. Masukkan **Password**.
4. Klik tombol **Login**.

> **Lupa Password?**
> Hubungi Administrator Sekolah atau Wali Kelas untuk melakukan reset password.

---

## 2. Navigasi Dashboard

Setelah berhasil login, Anda akan diarahkan ke Dashboard utama. Tampilan dashboard berbeda-beda tergantung peran (Role) Anda.

### Menu Utama (Sidebar Kiri)
- **Dashboard**: Ringkasan data (Siswa, Guru, Keuangan).
- **Akademik**: Data Siswa, Kelas, Nilai, Rapor.
- **SPMB**: Penerimaan Siswa Baru.
- **Perpustakaan**: Peminjaman buku.
- **Inventaris**: Aset sekolah.
- **Keuangan**: Tagihan siswa (SPP), Gaji Guru.
- **Pengaturan**: Profil sekolah, User Management.

### Profil Pengguna (Pojok Kanan Atas)
Klik nama/foto Anda untuk melihat menu profil:
- **Profil Saya**: Ubah data diri, foto profil.
- **Ganti Password**: Keamanan akun.
- **Keluar (Logout)**: Mengakhiri sesi.

---

## 3. Peran Pengguna (Role-Based Access)

Sistem ini membagi akses berdasarkan peran pengguna:

### 👑 Super Admin & Admin
- **Akses Penuh**: Dapat mengakses semua menu dan fitur.
- **Tugas Utama**: Mengelola User, Konfigurasi Sekolah, Backup Data.

### 👨‍🏫 Guru (Teacher)
- **Akses**: Akademik (Kelas, Nilai, Presensi), Perpustakaan (Pinjam Buku), Keuangan (Lihat Gaji).
- **Wali Kelas**: Memiliki akses tambahan untuk mengelola data siswa di kelasnya.

### 📚 Pustakawan (Librarian)
- **Akses**: Modul Perpustakaan (Katalog, Anggota, Sirkulasi, Denda).
- **Tugas Utama**: Mengelola buku dan pelayanan perpustakaan.

### 📦 Staff Sarpras (Inventory Staff)
- **Akses**: Modul Inventaris (Aset, Ruangan, Opname).
- **Tugas Utama**: Mengelola barang inventaris sekolah.

### 🎓 Siswa (Student)
- **Akses Terbatas**:
  - Melihat Profil Diri & Nilai.
  - Mencari Buku di Perpustakaan.
  - Melihat Tagihan Sekolah.
- **Login**: Menggunakan NIS/NISN sebagai Username.

---

## 4. Fitur Umum Lainnya

### 🔔 Notifikasi
Ikon lonceng di atas menampilkan pemberitahuan penting (Pengumuman Sekolah, Jatuh Tempo Peminjaman Buku, Tagihan SPP).

### 🔍 Pencarian Global
Gunakan kolom pencarian di bagian atas untuk mencari Data Siswa, Buku, atau Menu dengan cepat.

### 🌙 Dark Mode
Klik ikon Bulan/Matahari di menu profil untuk mengubah tampilan menjadi Gelap (Dark Mode) atau Terang (Light Mode).

---

## 5. Bantuan & Support

Jika Anda mengalami kendala teknis atau menemukan bug (kesalahan sistem):

1. Cek halaman **Bantuan** di menu profil.
2. Hubungi Tim IT Sekolah melalui email: `it-support@sekolahku.id`.
3. Sertakan screenshot (tangkapan layar) masalah yang Anda hadapi.
