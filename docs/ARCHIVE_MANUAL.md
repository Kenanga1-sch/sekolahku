# 📂 Panduan Pengguna: Arsip Digital & Alumni

Modul Arsip Digital digunakan untuk mengelola surat masuk, surat keluar, dan data alumni secara terpusat.

## 1. Surat Masuk (Incoming Mail)

Fitur ini digunakan untuk mencatat dan mendigitalisasi setiap surat yang diterima oleh sekolah.

### Mencatat Surat Masuk
1. Masuk ke menu **Arsip** > **Surat Masuk**.
2. Klik tombol **+ Catat Surat Baru**.
3. Isi formulir:
   - **Nomor Agenda**: Nomor urut pencatatan (Otomatis/Manual).
   - **Nomor Asli Surat**: Nomor yang tertera pada fisik surat.
   - **Pengirim**: Instansi atau perorangan yang mengirim.
   - **Perihal**: Ringkasan isi surat.
   - **Tanggal Terima**: Tanggal surat diterima oleh Tata Usaha.
   - **Klasifikasi**: Kode klasifikasi arsip (jika ada).
4. **Upload Scan Surat**: Unggah file PDF/Gambar surat asli untuk arsip digital.
5. Klik **Simpan**.

### Disposisi Surat
Setelah surat dicatat, status awalnya adalah `Menunggu Disposisi`.
1. Kepala Sekolah/Admin membuka detail surat.
2. Klik menu **Disposisi**.
3. Pilih penerima disposisi (Misal: Waka Kurikulum, Guru BK).
4. Tuliskan instruksi/catatan.
5. Status surat berubah menjadi `Terdisposisi`.

---

## 2. Surat Keluar (Outgoing Mail)

Fitur ini digunakan untuk manajemen penomoran dan arsip surat yang dikeluarkan sekolah.

### Membuat Surat Keluar / Booking Nomor
1. Masuk ke menu **Arsip** > **Surat Keluar**.
2. Klik tombol **+ Buat Surat / Booking Nomor**.
3. Isi formulir:
   - **Tujuan**: Penerima surat.
   - **Perihal**: Isi ringkas surat.
   - **Tanggal Surat**: Tanggal penerbitan.
   - **Klasifikasi**: Kode klasifikasi untuk penomoran otomatis.
4. Sistem akan memberikan **Nomor Surat** secara otomatis sesuai format sekolah.
5. Status awal adalah `Draft`.

### Upload Arsip Surat Keluar
Setelah surat fisik ditandatangani dan dicap:
1. Scan surat tersebut.
2. Buka detail surat keluar di sistem.
3. Upload file hasil scan.
4. Ubah status menjadi `Selesai` atau `Arsip`.

---

## 3. Arsip Alumni

Modul ini digunakan untuk menyimpan data siswa yang telah lulus untuk keperluan legalisir atau pelacakan studi lanjut.

### Menambah Data Alumni
Data alumni biasanya otomatis masuk saat proses **Kelulusan Siswa** di menu Akademik. Namun, admin bisa menambahkannya secara manual (untuk alumni lama).

1. Masuk ke menu **Arsip Alumni**.
2. Klik **+ Tambah Alumni**.
3. Isi data:
   - **Nama Lengkap & NISN**.
   - **Tahun Lulus**.
   - **Kelas Terakhir**.
   - **Sekolah Lanjutan/Pekerjaan** (Jejak Alumni).
4. Klik **Simpan**.

### Pencarian & Filter
- Gunakan kolom pencarian untuk mencari berdasarkan **Nama** atau **NISN**.
- Gunakan filter **Tahun Lulus** untuk melihat daftar alumni pada angkatan tertentu.

### Edit & Hapus
- Klik tombol titik tiga (menu) di sebelah kanan baris nama alumni.
- Pilih **Edit** untuk memperbarui data (misal: update sekolah lanjutan).
- Pilih **Hapus** jika data duplikat atau salah input (Hati-hati, data tidak bisa dikembalikan).

---

## 4. Laporan Arsip

Anda dapat mencetak laporan rekapitulasi surat:
1. Masuk ke menu **Arsip** > **Laporan**.
2. Pilih periode tanggal.
3. Pilih jenis laporan (Masuk/Keluar).
4. Klik **Cetak PDF** atau **Export Excel**.
